# Winson Scanner Config Codes

Bộ mã cấu hình quét trực tiếp cho máy quét Winson, trích từ manual gốc của dòng `WAI-5780`.

## Files

- `winson-config-page-1.png`
- `winson-config-page-2.png`
- `winson-config-page-3.png`
- `winson-config-codes-all.pdf` (bản gộp 3 trang để in)

## How To Use

- Mở trực tiếp file PNG/PDF trên màn hình và quét bằng máy.
- Nếu in giấy: in 100% scale (không fit, không shrink), giấy A4.
- Nên tăng sáng màn hình khi quét từ monitor.

## Notes

- Dòng `WAI-5770-USB` và `WAI-5780` cùng nhóm giao tiếp (`USB Keyboard`, `USB COM`, `RS232`, `USB HID`) nên có thể dùng chung bộ này để chuyển mode.
- Nếu model/firmware có khác biệt, ưu tiên mã trong manual đúng model.
